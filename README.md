# mymem スライドプレゼンテーション

「思考の手順を、すべての人に。」をテーマに、mymemの思想とビジョンを伝えるプレゼンテーションスライドです。

## スライド構成

### 1ページ目：タイトルスライド
**タイトル:** 思考の手順を、すべての人に。

**内容:**
- メインキャッチコピー「思考の手順を、すべての人に。」（青文字）
- サブコピー「私たちが作っているもの」
- プロダクト名「mymem」（グレー文字）
- 3つのロゴアイコン（アプリロゴ、マスコットキャラクター、ロゴマーク）

**デザイン要素:**
- 背景：複数の波線と青い円形ドット
- 波線は思考の流れを象徴

---

### 2ページ目：定義
**タイトル:** 定義 ー 「人とAIのあいだをつなぐ仕組み」

**内容:**
- 左側：メインメッセージ「AIが賢くなるほど、人とAIの意図はすれ違う。」（青文字）
- サブメッセージ「mymemは、人間の創造を再び中心に取り戻すための仕組み。」
- 右側：ベージュ色の角丸ボックス内にイラスト

**イラスト:**
- 左：人間（手描き風の棒人間）
- 右：AI/ロボット（手描き風の幾何学的図形）
- 中央：点線と疑問符で両者のすれ違いを表現
- 双方向の矢印

### 10ページ目：活用例
**タイトル:** 活用例（Use Cases）

**内容:**
- メインメッセージ「あらゆる専門家の"思考手順"を再利用できる」（青文字）
- サブメッセージ「スキルの再利用が、新しい創造の起点になる」

**3カラム構成:**

1. **🧠 開発者の場合**
   - Next.jsで多言語対応サイトを作りたい
   - → プロのワークフローを検索・クローン
   - → 自分用に再構成
   - 💬 初心者でもプロの思考手順を再現できる。

2. **🎓 教師・教育者の場合**
   - AIリテラシー授業構成を考えたい
   - → 世界の教師の授業設計スキルを検索
   - → 自クラスに最適化
   - 💬 教育の"再現知"が生まれる。

3. **🎨 クリエイターの場合**
   - 広告コピーを書きたい
   - → 共感ワード抽出スキルを検索
   - → ブランドに合わせ再構成
   - 💬 思考の再利用で創造スピードが爆上がり。

**背景:** 淡いブルーグラデーションとネットワークパターン

### 11ページ目：大切にしていること
**タイトル:** 大切にしていること（What We Care About）

**内容:**
- メインメッセージ「AIは「提案」するが、<br>「確定」するのは常に人間」（青文字、2行表示）

**フロー図:**

**下部注釈:**

### 13ページ目：製品ラインナップ
**タイトル:** 製品ラインナップ(Product Lineup)

**内容:**
- タイトル「思想を、かたちに。」（青文字）
- サブタイトル「mymem を構成する4つのプロダクトと、その進化。」

**4列構造図:**

1. **mymem Lite** 
   - アプリ・体験層
   - 一般ユーザー向け体験層。Chrome拡張・ローカル記憶AI
   - 想定:一般利用者

2. **mymem Pro**
   - 開発・統合層
   - AIとの記憶を個人の主権とする(Pro)
   - 想定:開発者・一般利用者

3. **mymem Enterprise**
   - 企業統合層
   - 大規模組織向けの知識統合・協働基盤
   - 想定:エンタープライズ

4. **mymem OS**
   - 思想・基盤層
   - 思考構造と知の再現性を支える思想・基盤
   - 想定:経営層・研究層

**タイムライン(Phase 1-4):**
- **Phase 1:** mymem Lite(リリース済 2025)
  - ローカル記憶AI(Chrome拡張)
  - 思考と記憶の統合
- **Phase 2:** mymem Pro(2026年4月リリース予定)
  - AI・人間・ツールを繋ぐワークフローAI基盤とクラウド統合
  - 自動学習・自己拡張
- **Phase 3:** mymem Enterprise(計画中 2026)
  - 組織知の統合と協働基盤
  - 大規模チーム連携
- **Phase 4:** mymem OS(2027年から)
  - 思考の構造そのものを司る知的基盤
  - AIと人の共創インフラ

## デザインの特徴

### カラースキーム
- **メインブルー:** #2A6DF5, #2563EB（キーワード、見出し）
- **グレー:** #1F2937, #374151, #6B7280（本文、説明文）
- **ベージュ:** #F5F1E8（イラストボックス背景）
- **淡いブルー:** #F8FAFC, blue-50（背景）

### タイポグラフィ
- 見出し：大きく太字、青色で強調（text-3xl〜text-5xl）
- 本文：読みやすいグレー、適度な行間
- 重要キーワードは青色で統一
- 左上セクションタイトル：text-3xl、text-gray-800

### イラストスタイル
- シンプルで素朴な手描き風
- フリーハンドのような線と図形
- 有機的で温かみのある表現
- 思考の流れを波線や矢印で視覚化
- ネットワークパターンで接続性を表現

### レイアウト
- 左右2カラムレイアウト（テキスト + 図）
- 中央揃えの強調メッセージ
- 余白を活かしたミニマルなデザイン
- 一貫性のあるセクションタイトル配置（左上、p-16エリア内）
- 3〜4カラムのカードレイアウト（活用例、製品ラインナップ）
